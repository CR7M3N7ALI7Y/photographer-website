## Структура сайта
1. **Шапка** — логотип и адаптивное навигационное меню
2. **Приветственный блок** — фоновое изображение с призывом к действию
3. **Портфолио** — галерея из 6 работ с подписями
4. **Услуги** — 3 карточки с ценами и описанием
5. **Контакты** — контактная информация и форма обратной связи
6. **Подвал** — копирайт и ссылки на соцсети

## Используемые технологии
